## 💻 Sobre o projeto

✍️ Feedback UI - é uma web aplicação que permite o usuario escrever reviews .

---

## ⚙️ Funcionalidades

- [x] Usuario pode criar as Reviews.
- [x] Usuario pode Deletar as Reviews.
- [x] Usuario pode Editar as Reviews.
- [x] Dados são salvos no Json DB.
